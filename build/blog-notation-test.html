ブログ記法のテスト

2019-11-17

新しく作ったブログ記法のテストを行っておく。関数をかたっぱしから呼んでいってうまく動いているかを試す。

inline系の関数

引数の中身を解釈する関数

先ずは@a

これはリンクです。

次に@rel

この記事のパスは../build/blog-notation-test.htmlです。

引数の中身を解釈しない関数

次は@incode

これはa = 1です。

次は@inmath

アインシュタインは$E = mc^2$を発見しました。

ブロック系の関数

引数の中身を解釈する関数

次に@ul

  • アイテム
  • その他のアイテム

次に@ol

  1. 一つ目のアイテム
  2. 二つ目のアイテム

次に@def

定義名

定義の内容一行目

定義の内容二行目

次に@fig

次に@table

行頭1行頭2
列1の1つめ列1の2つ目
列2の1つめ列2の2つ目

次は@quote

自分で、すぐ自分を許せる人間は、大した仕事をやらない。

引数の中身を解釈しない関数

次は@blcode

#include <iostream>

int main() {
  std::cout << "Hello World" << std::endl;
}

次は@blmath

$$e^{i\pi} + 1 = 0$$
