Rのパッケージimagerを入れる

2018-12-10

適当にRのパッケージであるimagerをインストールしようと思ったらできなかったのでメモ。

とりあえずGoogleで検索するとちょうど同じ問題に突き当たっていた人がStackoverflowで質問をしていた。解答によればCairoパッケージを入れればいいらしい。とりあえずbrewで入れてみる

$ brew install cairo

と何の問題もなく入ったので、とりあえず再度インストール

$ r
> install.packages("imager")

今度はちゃんと入ってくれた。
